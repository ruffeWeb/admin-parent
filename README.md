# SpringBootAdmin
> 说明

此项目是SpringBoot搭建的后台基础框架，用的H+后台模板和Freemark模板引擎，集成了Redis，ehcache，Mencached缓存以及MQ，使用Fremark自定义标签实现了按钮权限（这里么有使用Shiro），实现了微信授权等操作，只需要更改application.propertiesd中的 ***  符号。
